Promises in ECMAScript 6

In ECMAScript 6, promises are a powerful way to handle asynchronous operations. Promises allow you to write cleaner and more manageable code when dealing with operations that take time, such as network requests or timers.

Here is an example of how to use promises in ECMAScript 6:

let value = 0;
function fun1() {
  return new Promise(function (resolved) {
    setTimeout(function () {
      resolved(value + 1);
    }, 1000);
  });
}
